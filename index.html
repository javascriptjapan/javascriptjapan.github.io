Tools

JavaScriptでアプリを作成するときに役立つ便利なツールがたくさんあります。
各種ツールの使い方もマスターして活用していきましょう！

Visual Studio Code

Microsoft社で開発されている無料のIDEです。 各種のプラグインによって機能を拡張できます。 JavaScriptのアプリ開発で使ってみましょう！

Git

無料のバージョン管理ソフトです。 何度でも自由に作り直せる環境を用意できます。 試作品をたくさん作ってみましょう！

GitHub

自分が作った作品をオープンソースで公開できます。 GitHub Pagesを使えば無料でWebページを作成できます。

Docker

サーバーを仮想化する技術です。 開発環境を簡単に用意できるので便利です。

Vue.js

JavaScriptのフレームワークです。 使い方が簡単なので、jQueryの代わりに使ってみましょう！

React.js

Facebook社で開発されているJavaScriptのフレームワークです。 React Nativeなどスマホアプリも作れるので便利です。
